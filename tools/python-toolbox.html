Amazon SellerSprite MCP Skill Python Toolbox

Python + HTML 辅助入口，保留原 PowerShell 脚本作为备份与兼容方案。

先在仓库根目录运行命令。Windows 如未配置 PATH，可把 python 替换为实际解释器路径，例如 %LOCALAPPDATA%\Python\pythoncore-3.14-64\python.exe。

完整包检查

执行脱敏扫描、必需文件检查、Excel XML 完整性检查。

命令
python scripts/python/sellersprite_skill_tools.py check-package

敏感信息扫描

扫描 API key、GitHub token、真实 secret URL、本地路径、真实 ASIN 样式。

命令
python scripts/python/sellersprite_skill_tools.py scan-sensitive

运行配置检查

检查 `.env`、环境变量和 `.mcp.json` 是否已配置，不打印真实 key。

命令
python scripts/python/sellersprite_skill_tools.py check-runtime-config --strict

Skill 逻辑检查

检查所有 open-source skill 是否包含凭证预检、SellerSprite MCP 与 Gemini/GLM key 处理说明。

命令
python scripts/python/sellersprite_skill_tools.py test-open-source-skills

安装到 Codex

把 `skills/` 复制到当前用户的 `.codex/skills`。

命令
python scripts/python/sellersprite_skill_tools.py install-skill --target codex

发布到 GitHub

使用 GitHub Contents API 更新仓库；需要当前 shell 临时设置 `GITHUB_TOKEN`。

命令
python scripts/python/sellersprite_skill_tools.py publish-github-api --repo-name amazon-sellersprite-research-MCP-skill
